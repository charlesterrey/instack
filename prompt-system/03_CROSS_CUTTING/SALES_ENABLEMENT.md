### 5 étapes :
1. **Intelligence** : Identifier le DSI, mapper l'org, comprendre les enjeux IT
2. **Ally** : Trouver un champion interne (Ops Manager qui utilise déjà instack)

## BATTLE CARDS

### vs Power Apps
- NPS Power Apps : -24.1 (source : peer reviews)
- 40-60% apps orphelines
- Time to first app : 3-6 mois vs 90 secondes
- Coût : 20-40 EUR/user/mois vs 5 EUR/creator/mois
- "Power Apps est fait pour les devs. instack est fait pour les Ops."

### vs Retool / Appsmith
- ICP différent : Retool = développeurs, instack = métiers
- Retool : 10 EUR/user/mois, nécessite du code
- instack : 0 code, IA contrainte, apps jetables

### vs Shadow IT (statu quo)
- 97% des apps cloud inconnues de l'IT
- Coût moyen Shadow IT : 21M USD/an pour un grand groupe
- instack : visibilité totale + contrôle sans friction

### vs v0/bolt.new (AI builders)
- 91.5% des apps IA ont des vulnérabilités
- instack : génération contrainte, zero code libre
- "Ils génèrent du code. Nous assemblons des composants sécurisés."
